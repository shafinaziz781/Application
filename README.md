🍽️ **Food Ordering System**  
A modern desktop-based **Food Ordering System** built using **Java Swing.** <br>
This application allows users to log in, select a restaurant, browse menu items, place orders, and view a detailed receipt. <br>

This is a basic sample project designed to demonstrate core Object-Oriented Programming (OOP) concepts in Java, including abstraction, inheritance, polymorphism, and encapsulation.  <br>

It serves as a practical implementation to showcase how these fundamental principles are applied in building a structured desktop application. <br>

🚀 **Features**
•	🔐 User Login System (Default credentials included). <br>
•	🏪 Multiple Restaurants Selection. <br> 
•	📋 Dynamic Menu System. <br> 
•	➕ Add / ➖ Remove Items from Cart. <br> 
•	🛒 Live Cart Update. <br> 
•	💰 Automatic Total Calculation. <br> 
•	🧾 Receipt Generation with GST. <br> 
•	🎨 Modern Dark UI (Custom Styled). <br> 
•	🖱️ Interactive Buttons & Hover Effects. <br>
🏗️ **Project Structure**


Application.java

│
├── Login (Login Screen)

├── RestaurantSelection (Restaurant chooser UI)

├── OrderScreen (Menu & Cart system)

├── ReceiptScreen (Final bill display)


├── Restaurant (Abstract Base Class)

│   ├── McDonalds

│   ├── Howdy

│   └── Ranchers

└── UIHelper (UI Styling Utilities) <br>

🧠 **Concepts Used.** <br>

**🔹 Object-Oriented Programming** (OOP)<br>

•	Abstraction (abstract class Restaurant).

•	Inheritance (McDonalds, Howdy, Ranchers).

•	Polymorphism (method overriding).

•	Encapsulation (data handling in classes). <br>

**🔹 Java Swing** (GUI)

•	JFrame, JPanel, JLabel, JButton. 

•	Layout Managers (BorderLayout, BoxLayout, FlowLayout). 

•	Event Handling (ActionListener, MouseListener).

**🔹 Data Structures**<br>

•	Map (LinkedHashMap for menu & cart). <br>

•	Dynamic item tracking using key-value pairs. <br>

**🔹 Custom UI Design**<br>

•	Rounded panels. <br>

•	Custom painted buttons. <br>

•	Dark theme styling. <br>

•	Emoji-based UI elements. <br>
